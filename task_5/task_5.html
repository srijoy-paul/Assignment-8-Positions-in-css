About CSS z-index Property

z-index is a CSS property that determines the stacking order of positioned elements on the z-axis (the axis coming out of the screen toward the viewer). It is commonly used to control the visual stacking of overlapping elements. The element with a higher z-index value is displayed in front of elements with lower z-index values.

Examples

In example 1, box2 has higher z-index. So it is stacking over box1.

In example 2, box3 has higher z-index. So it is stacking over box4.
